# Casa dos Parafusos

Bem-vindo ao projeto **Casa dos Parafusos**! Este é um site fictício para uma loja de ferramentas e ferragens localizada em Porto Ferreira-SP, desenvolvido com HTML, CSS e JavaScript.

## Sobre o Projeto

A **Casa dos Parafusos** é uma loja tradicional, há mais de 20 anos no mercado, oferecendo alicates para crimpagem, brocas, parafusos, ferragens e soluções de fixação com qualidade e preço justo. Este projeto tem como objetivo criar uma presença online para a loja, apresentando suas categorias de produtos, diferenciais e informações para contato.

## Funcionalidades

- **Página Inicial:** Apresenta a loja, história, produtos principais e chamada para contato.
- **Slider de Produtos:** Visualização organizada das categorias de produtos e soluções oferecidas pela loja.
- **Menu Responsivo:** Interface adaptada para desktop e dispositivos móveis, com menu hamburguer.
- **Solicitação de Orçamento:** Botão de contato rápido via WhatsApp para facilitar orçamento e dúvidas.
- **Footer Informativo:** Contém informações de contato, horário de funcionamento, redes sociais e breve descrição sobre a loja.

## Instalação e Uso

1. Clone o repositório:  
   `git clone https://github.com/dsm-fatec-2025/casa-dos-parafusos.git`
2. Abra o arquivo `index.html` em um navegador da web.
